How to push to remote repository?

To push to a remote repository, follow these steps:

  1. Ensure you have a remote repository set up (e.g., on GitHub).
  2. Initialize a local repository using git init.
  3. Add your files to the staging area using git add ..
  4. Commit your changes with git commit -m "Your commit message".
  5. Link your local repository to the remote repository using git remote add origin <repository-URL>.
  6. Push your changes to the remote repository using git push -u origin main (replace main with your branch name if different).
